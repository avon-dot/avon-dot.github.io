MacOS安装MySQL

发表于 2023-03-03 分类于 数据库 阅读次数：

MySQL的安装包列表可以查看 https://www.mysql.com/support/supportedplatforms/database.html
可以选择合适自己的系统的安装包进行安装

MacOS的安装形式有几种

  • 通过dmg安装包安装
  • 二进制原码安装

下载相应的包，进行安装

下载之前可以通过命令查看自己系统内核版本，选择合适的版本下载安装

安装

双击下载的dmg文件，按照步骤安装即可，中间需要输入密码之类的。

注意
在安装MySQL之前需要关闭正在运行的MySQL服务实例，不管是用的客户端启动的服务还是命令行启动的服务，都需要关闭

具体的安装步骤解释可以看文档

注意中间需要设置输入root的密码

安装路径

安装结束之后，可以查看/usr/local/mysql这个路径就是安装的路径，实际指向的具体的版本信息

安装的文件夹的含义

运行MySQL

安装好了之后，可以直接运行mysql

为了方便运行mysql，我们可以设定一下当前的环境变量

mac现在的终端用的zsh，修改.zshrc文件即可，有的用的是bash，修改.bash_profile文件，改完之后source ~/.zshrc即可生效，可以直接使用
